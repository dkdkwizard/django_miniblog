django_miniblog
====
用 Django 實作的部落格
連結:
https://miniblog-72415.herokuapp.com/blog/

提供基礎部落格功能，包含:註冊、發文、回應、文章分類、人次計算及熱門文章推薦。

使用 Heroku 雲端架設網站並以 AWS S3 儲存影像資料。
